Servidor central de colaboradores do Skore Manager.

Uso local:

- `npm run server`
- API em `http://localhost:3030`
- Socket.IO no mesmo host

Endpoints:

- `GET /health`
- `GET /api/collaborators`
- `POST /api/collaborators`
- `PATCH /api/collaborators/:matricula`
- `DELETE /api/collaborators/:matricula`
